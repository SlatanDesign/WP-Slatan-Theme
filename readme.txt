== Description ==

WP Slatan Theme is a blazing-fast, secure WordPress theme built specifically for Elementor. 
Based on the Underscores (_s) starter theme, it provides a rock-solid foundation for building 
beautiful websites with Elementor.

= Key Features =

* **100% Elementor Compatible** - Full support for Elementor and Elementor Pro Theme Builder
* **Header/Footer Builder Support** - Use Elementor Pro to build custom headers and footers
* **Lightweight** - Minimal CSS (~180 lines) for faster loading
* **Secure** - Built-in security headers, XML-RPC toggle, login protection
* **Performance Optimized** - No jQuery Migrate, deferred scripts, no emoji bloat
* **Control Panel** - Easy-to-use theme settings page

= Page Templates =

* **Elementor Canvas** - Blank page template, perfect for landing pages
* **Full Width** - Header and footer, but no sidebar

= Shortcode =

Use saved Elementor templates anywhere with:

`[wpslt_template id="123"]`

== Installation ==

1. Upload the `wp-slatan-theme` folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance' menu in WordPress
3. Install and activate Elementor (free) or Elementor Pro for full functionality
4. Configure theme settings in Appearance â†’ Theme Settings

== Frequently Asked Questions ==

= Do I need Elementor Pro? =

The free version of Elementor works great for page content. Elementor Pro is needed 
for the Header/Footer Builder feature.

= Is this theme SEO friendly? =

Yes! The theme outputs clean, semantic HTML and is fully compatible with all major SEO plugins.
